Программа Paint.Net
Окно инструментов 

В Окне инструментов выбираются инструменты.
Нажатие на значок инструмента делает этот инструмент active и доступным для редактирования изображения.









Инструменты выделения

Прямоугольник 
Инструмент для выделения прямоугольной или квадратной области. 

Лассо
Инструмент для выделения области свободной формы.

Эллипс
Инструмент для выделения круглой или эллиптической области.

Волшебная палочка
Для выделения областей активного слоя, близких по цвету.

Инструменты перемещения

Перемещение выделенной области
Используется для перемещения пикселей, которые находятся в слое или текущем выделении.

Перемещение области выделения
Используется для перемещения или изменения контура выделения, не затрагивая пиксели изображения.

Инструменты просмотра

Масштаб
Используется для увеличения, уменьшения или расширения полотна вокруг определенной области.

Рука
Выполняет прокрутку или перетаскивание изображения. Эта функция может также быть использована из любого другого инструмента в любое время: удерживайте клавишу пробела, а затем нажмите и перетащите мышью.

Инструменты заливки

Заливка
Заполняет области одинакового цвета другим цветом.

Градиент
Позволяет рисовать градиенты (постепенный переход одного цвета в другой). Поддерживается несколько форматов и форм. Также есть режим прозрачности, который полезен для "осветления", а также для "смешивания" двух и более изображений вместе.

Инструменты рисования

Кисть
Это инструмент по умолчанию при запуске paint.net. Он используется для создания полос цвета, как это делает кисточка с краской, отсюда и название.

Ластик
Как следует из названия, этот инструмент "стирает" выбранные области изображения, устанавливая непрозрачность, или альфу равной нулю.  

Карандаш
Позволяет редактировать активный слой попиксельно.

Инструменты для фото

Пипетка
"Принимает" цвет от активного слоя и устанавливает его как текущий Основной или Дополнительный цвет. Подробное описание смотрите в разделе Пипетка.

Клонирующая кисть
Используется для копирования областей пикселей между разными слоями или внутри одного слоя. 

Замена цвета
Заменяет один цвет другим. 

Текст и инструменты формы

Текст
Размещает текст на активном слое. 

Линия/Кривая
Рисует прямые или изогнутые линии на активном слое. 

Фигуры
Используется для рисования ряда предопределенных фигур.
